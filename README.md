\# E-commerce Backend API



\## Tech Stack

\- Django REST Framework

\- MySQL



\## Features

\- JWT Authentication

\- Role-based access (Admin/User)

\- Product management

\- Order system



\## API Endpoints

\- /api/auth/login

\- /api/products

\- /api/orders



\## Description

A RESTful API for managing an e-commerce platform with authentication and order processing.
